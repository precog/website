Precog works alongside your existing databases to store event-oriented (transactional) data. If you have transactional data stored in your existing databases, it's easy to transfer it to Precog with our REST API, or you can browse the add-on marketplace for apps that can automatically synchronize your data to Precog.

Precog's JavaScript client library makes it easy to capture data directly from web applications. User activity, such as engagement, page views, clicks, purchases, conversations, shares, or other actions relevant to your app can all be captured and stored in Precog.

Precog's REST API and many client libraries (PHP, Ruby, Java, Python) allow you to capture data that originates from your servers, or transfer data already stored on your servers. Log files, purchase events, IT events, file downloads, and all other transactional data can be captured and stored in Precog.

Precog's REST API makes it easy to capture data from third-party sources, such as external APIs (Twitter, Facebook, SalesForce), CSV files, and websites. The add-on marketplace contains a growing collection of add-ons that can import data from all across the web.

Precog's REST API makes it easy to capture data that originates on a mobile device, from app opens, to user interactions, user comments, and more. Look for our upcoming iOS and Android APIs to make integration even easier.

The add-on marketplace contains many demographic enrichments that let you enrich data (emails, locations, or IPs) with rich demographic information that can power advanced analytics, customer segmentation, recommendations, and much more.

The add-on marketplace contains a variety of sentiment analysis add-ons that let you determine the sentiment contained inside text written in most languages (overall sentiment or sentiment directed at particular persons, places, or things).

The add-on marketplace contains add-ons that can enrich location data with addresses, places, people, and events, and add-ons that can enrich an IP address with location information or clean-up a partial address.

The add-on marketplace contains a facet add-on that can enrich any person, place, or thing with facets (such as the age of a person or the number of employees of a company), for more powerful analysis.

The add-on marketplace has a weather add-on that can enrich your location / time data with detailed historical weather data.

These add-ons are just a small sampling of the add-ons in the marketplace. Check it out today to learn how you can take your data, and make it far more valuable through third-party data and computational enrichments.
