Overview

Sending event data from the browser is a snap, thanks to the ReportGrid core JS API. Simply include the reportgrid-core.js in your HTML document's head with your tokenId appended to the URL, and use the ReportGrid global's local functions to AJAX your calls to the REST API

<!DOCTYPE html>
<html>
  <head>
    <title>MySite.com</title>
    <script type="text/javascript" src="http://static.reportgrid.com/api/js/v1/reportgrid-core.js?
       tokenId=[YOUR TOKEN ID]"></script>
  </head>
  <body>

    <script type="text/javascript">
      ReportGrid.track("/merchants/BrewersPlace/locations/USA_CO_Boulder/1/",
        {
          "events": {
            "purchase": {
              "item": "Americano",
              "size": "small"
            }
          }
        }
      );
    </script>
  </body>
</html>

ReportGrid Event Metric Retrieval

fieldCount(path, field, success, failure)

METHOD PARAMETERS
fieldCount path, field : [eventPath], success(v), failure(e)
Result
The total counts of how often the specified field appeared in events of the specified type.
EXAMPLE
Retrieve count of field delivery.status from path /account/jdoe/emails/ 
ReportGrid.fieldCount(
  "/account/jdoe/emails/",
  { "field" : "delivery.status" },
  alert,
  logger.error
);

fieldSeries(path, opt, success, failure)

METHOD PARAMETERS
fieldSeries path, opt : {field : fieldName, periodicity : selectedPeriodicity}, success(v), failure(e)
Result
The time series counts of how often the specified field appeared in events of the specified type.
EXAMPLE
Retrieve time series by day of field delivery.status from path /account/jdoe/emails/ 
ReportGrid.fieldSeries(
  "/account/jdoe/emails/",
  {
    "field" : "delivery.status",

  },
  toChart,
  logger.error
);

ReportGrid Utilities

METHOD PARAMETERS

Result
An array of children of the specified path.
EXAMPLE
Retrieve all the children of path customers 
ReportGrid.children(
  "/customers/",
  {"type":"all"},
  sortChildren,
  logger.error
);
